Proyectos

Una selección de proyectos de ciberseguridad e ingeniería de software que he construido, a los que he contribuido o en los que estoy trabajando actualmente.

Sitio Personal Lake & Ocean

2025 | Sitio Web Personal

Un sitio web personal bilingüe (EN/ES) construido con HTML, CSS y JavaScript vanilla puro. Presenta un sistema de diseño personalizado "Lake & Ocean" que conecta Miami y Chicago, con temas claro/oscuro, búsqueda de blog y soporte completo de accesibilidad. Sin frameworks, sin herramientas de compilación—solo código rápido y mantenible.

HTML CSS JavaScript Accesibilidad i18n

Plataforma de Inteligencia de Amenazas Impulsada por IA

2024-2025 | Investigación de Seguridad

Proyecto de investigación explorando el uso de Modelos de Lenguaje Grande para la recopilación automatizada de inteligencia de amenazas y análisis. Implementa RAG (Generación Aumentada por Recuperación) para mejorar la precisión de detección de amenazas y reducir falsos positivos en flujos de trabajo de operaciones de seguridad.

Python LLMs Inteligencia de Amenazas RAG Operaciones de Seguridad

Laboratorio de Capacitación en Ciberseguridad

2023-Presente | Educación

Un entorno de laboratorio virtual integral para enseñar conceptos de ciberseguridad. Construido sobre Docker y VirtualBox, proporciona experiencia práctica con seguridad de redes, pruebas de penetración, respuesta a incidentes y estrategias de defensa. Utilizado en más de 600 clases con documentación bilingüe.

Docker VirtualBox Redes Educación en Seguridad Infraestructura

Recursos de Seguridad Bilingües

2022-Presente | Código Abierto

Colección de materiales educativos de ciberseguridad traducidos al español. Incluye guías de estudio, hojas de referencia y ejercicios prácticos que cubren temas desde higiene de seguridad básica hasta caza avanzada de amenazas. Objetivo: hacer que la educación en seguridad sea accesible para las comunidades de habla hispana.

Educación Documentación Español Código Abierto Comunidad

Scripts de Automatización de Seguridad

2024 | DevSecOps

Colección de scripts de Python y Bash para automatizar tareas comunes de seguridad: análisis de logs, escaneo de vulnerabilidades, verificación de cumplimiento y generación de reportes de seguridad. Diseñados para integrarse con pipelines de CI/CD para monitoreo continuo de seguridad.

Python Bash Automatización DevSecOps CI/CD

Manuales de Respuesta a Incidentes

2023 | Documentación

Manuales integrales de respuesta a incidentes desarrollados a partir de experiencia en operaciones de seguridad del mundo real. Cubre ransomware, phishing, brechas de datos y amenazas internas con procedimientos paso a paso, árboles de decisión y plantillas de comunicación. Disponible en inglés y español.
